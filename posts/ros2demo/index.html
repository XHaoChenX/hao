ROS2 踩坑记录

April 6, 2026 · HaoChen

做这个 demo 目的只是为了看看 ROS1 和 ROS2 区别。

我自己的话其实只涉猎过 ROS1 和 CycloneDDS。

了解了发现其实 ROS2 底层 DDS 是可以切换的，可以是 CycloneDDS 也可以是 FastDDS。

ROS2 架构#

ROS2 通过 RMW 抽象层支持多种 DDS 实现：

ROS2 API
   ↓
RMW（抽象层）
   ↓
DDS实现（Cyclone / FastDDS / Connext / GurumDDS）
